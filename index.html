Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS provides the content style.
  • Margins indicate how much space we want around the outside of an element.
  • Padding indicates how much space we want around the content inside an element.

Git

  • Git is a version control system that lets you manage and keep track of each version of your project.
  • Allows developers and others to use version control.
  • git status gets the status of branch.

JavaScript

  • Handles logic and allows users to interact with a website.
